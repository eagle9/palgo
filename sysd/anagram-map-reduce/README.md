lint 503. Anagram (Map Reduce)
difficulty: medium

Use Map Reduce to find anagrams in a given list of words.

Example
Example 1:
Input: "lint lint lnit ln"
Output: 
  ["lint", "lint", "lnit"]
  ["ln"]

Example 2:
Input: "ab ba cab"
Output:
  ["ab", "ba"]
  ["cab"]
