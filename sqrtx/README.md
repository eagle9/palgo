141. Sqrt(x)
Description
Implement int sqrt(int x).

Compute and return the square root of x.

Have you met this question in a real interview?  
Example
sqrt(3) = 1

sqrt(4) = 2

sqrt(5) = 2

sqrt(10) = 3
